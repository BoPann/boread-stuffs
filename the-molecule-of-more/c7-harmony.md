
# Chapter 7 Harmony

最重要的一個章節，如何與多巴胺 h&n 共處？如何創造一個成功的人生，又能快樂的生活？

過多的多巴胺(工作狂）或過多的h&n（嬉皮？）都不好，平衡兩者之間的參與才是最好的方式。

1. 找到一個自己做得非常好的的技能，或是去培養這個技能。 在學習或是練習的過程需要多巴胺的參與。而學成後的成就感就是h&n。

2. 感受當下，因為在當下，每個下一個瞬間都是驚喜。利用prediction error 去創造多巴胺的分泌。 而在感受當下的同時也需要個種感官的參與 -> h&n

3. 專注在做一件事情。

4. 創造。 做一些很基本但有創造力的東西像是裁縫 木工 畫畫等等。

多巴胺也許能驅動我們，追求更好的生活，讓我們在這個世界生存。但感到快樂和滿足，我們需要 h&n 的參與。
